Universal Quantifiers

  • Quantifiers are words that refer to quantities such as ”some” or ”all” and tell for how many elements a given predicate is true.
  • The symbol ∀ denotes ”for all” and is called the universal quantifier.
  • Let Q(x) be a predicate and D the domain of x. A universal statement is a statement of the form ”∀x ∈ D, Q(x).” It is defined to be true if, and only if, Q(x) is true for every x in D. It is defined to be false if, and only if, Q(x) is false for at least one x in D. A value for x for which Q(x) is false is called a counterexample to the universal statement. For Example: We can express the sentence ”All human beings are mortal,” as ∀ human beings x, x is mortal. 1

Existential Quantifiers

  • The symbol ∃ denotes ”there exists” and is called the existential quantifier.
  • Let Q(x) be a predicate and D the domain of x. An existential statement is a statement of the form ”∃x ∈ D such that Q(x).” It is defined to be true if, and only if, Q(x) is true for at least one x in D. It is false if, and only if, Q(x) is false for all x in D. For Example: ∃m ∈ Z + such that m2 = m.
